# `@base-ui/lit`

This package is the work-in-progress Lit port of the `@base-ui/react` surface.

Current status:

- package scaffold exists
- migration inventory is tracked in [`migration-inventory.json`](./migration-inventory.json)
- no public Lit component implementations have been ported yet

The migration goal is to mirror the public `@base-ui/react` export surface with Lit equivalents rather than creating an unrelated API.
