## Anatomy

```html title="Anatomy"
<accordion-root>
  <accordion-item value="item-1">
    <accordion-header>
      <accordion-trigger>Section title</accordion-trigger>
    </accordion-header>
    <accordion-panel>Section content</accordion-panel>
  </accordion-item>
</accordion-root>
```
