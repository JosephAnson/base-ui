## Anatomy

Import the component and assemble its parts:

```html title="Anatomy"
<accordion-root>
  <accordion-item value="item-1">
    <accordion-header>
      <accordion-trigger />
    </accordion-header>
    <accordion-panel />
  </accordion-item>
</accordion-root>
```
