## Anatomy

Import the custom elements and assemble the parts:

```html title="Anatomy"
<fieldset-root>
  <fieldset-legend>Notifications</fieldset-legend>
</fieldset-root>
```

## Usage guidelines

- Use `fieldset-root` to group related controls under a shared label.
- Use `fieldset-legend` as the accessible label for the group instead of separate heading-only text.

## API reference

### FieldsetRootElement

Custom element: `<fieldset-root>`

| Attribute  | Type      | Default | Description                                           |
| ---------- | --------- | ------- | ----------------------------------------------------- |
| `disabled` | `boolean` | `false` | Whether the component should ignore user interaction. |

### FieldsetLegendElement

Custom element: `<fieldset-legend>`

Accessible legend associated with the nearest `fieldset-root`.

### Data attributes

`<fieldset-root>` and `<fieldset-legend>` receive:

- `data-disabled`
