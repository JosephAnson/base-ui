## API Reference

### Root

Groups all parts of the field.
Renders a `<field-root>` custom element.

**Root Props:**

| Prop                   | Type                                                                                                                           | Default      | Description                                                                                                             |
| :--------------------- | :----------------------------------------------------------------------------------------------------------------------------- | :----------- | :---------------------------------------------------------------------------------------------------------------------- |
| name                   | `string`                                                                                                                       | -            | Identifies the field when a form is submitted.&#xA;Takes precedence over the generated control name.                    |
| actionsRef             | `{ current: Root.Actions \| null }`                                                                                            | -            | A ref to imperative actions.&#xA;`validate`: validates the field when called.                                           |
| dirty                  | `boolean`                                                                                                                      | -            | Whether the field's value has changed from its initial value.&#xA;Useful when the field state is controlled externally. |
| touched                | `boolean`                                                                                                                      | -            | Whether the field has been touched.&#xA;Useful when the field state is controlled externally.                           |
| disabled               | `boolean`                                                                                                                      | `false`      | Whether the component should ignore user interaction.&#xA;Takes precedence over `disabled` on the generated control.    |
| invalid                | `boolean`                                                                                                                      | -            | Whether the field is invalid.&#xA;Useful when the field state is controlled externally.                                 |
| validate               | `((value: unknown, formValues: Record<string, unknown>) => string \| string[] \| Promise<string \| string[] \| null> \| null)` | -            | A function for custom validation.&#xA;Return a string or array of strings when invalid, or `null` when valid.           |
| validationMode         | `ValidationMode`                                                                                                               | `'onSubmit'` | Determines when the field should be validated.&#xA;Takes precedence over the `validationMode` on `<form-root>`.         |
| validationDebounceTime | `number`                                                                                                                       | `0`          | How long to wait between validation callbacks when `validationMode="onChange"`.&#xA;Specified in milliseconds.          |

type RootApiProps = {
  /**
   * Identifies the field when a form is submitted.

   */

  /**
   * A ref to imperative actions.
   * `validate`: validates the field when called.
