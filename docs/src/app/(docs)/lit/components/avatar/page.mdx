## Anatomy

```html title="Anatomy"
<avatar-root>
  <avatar-image src="/avatar.jpg" alt="Ada Lovelace"></avatar-image>
  <avatar-fallback>AL</avatar-fallback>
</avatar-root>
```

## API reference

### Root

Custom element: `<avatar-root>`

Tracks shared image loading status for the image and fallback parts.

### Image

Custom element: `<avatar-image>`

| Property                | Type               | Default | Description                                     |
| ----------------------- | ------------------ | ------- | ----------------------------------------------- |
| `onLoadingStatusChange` | `(status) => void` | —       | Callback fired when the loading status changes. |

### Fallback

Custom element: `<avatar-fallback>`

| Attribute | Type     | Default | Description                                                    |
| --------- | -------- | ------- | -------------------------------------------------------------- |
| `delay`   | `number` | —       | How long to wait before showing the fallback, in milliseconds. |
