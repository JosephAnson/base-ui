## Anatomy

Use `radio-root` together with [`radio-group`](/lit/components/radio-group):

```html title="Anatomy"
<radio-group>
  <radio-root value="system">
    <radio-indicator></radio-indicator>
    System
  </radio-root>
</radio-group>
```

## API reference

### RadioRootElement

Custom element: `<radio-root>`

| Property                  | Type      | Default | Description                                       |
| ------------------------- | --------- | ------- | ------------------------------------------------- |
| `.value`                  | `unknown` | —       | Value used to identify this radio inside a group. |
| `disabled`                | `boolean` | `false` | Disables the radio.                               |
| `read-only` / `.readOnly` | `boolean` | `false` | Prevents the radio from being selected.           |
| `required`                | `boolean` | `false` | Marks the radio as required.                      |

`radio-root` renders a hidden native `<input type="radio">` internally for form participation and label association.

### RadioIndicatorElement

Custom element: `<radio-indicator>`

| Attribute / Property            | Type      | Default | Description                                              |
| ------------------------------- | --------- | ------- | -------------------------------------------------------- |
| `keep-mounted` / `.keepMounted` | `boolean` | `false` | Keeps the indicator mounted when the radio is unchecked. |

## Data attributes

`radio-root` and `radio-indicator` can receive:

- `data-checked`
- `data-unchecked`
- `data-disabled`
- `data-readonly`
- `data-required`
