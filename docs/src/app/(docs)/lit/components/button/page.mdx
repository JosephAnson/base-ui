## Usage guidelines

- `button-root` always enforces button semantics. It should not be used for links.
- Use `focusable-when-disabled` for loading buttons so focus stays on the control when it becomes disabled.

## Anatomy

Import the custom element:

```html title="Anatomy"
<button-root>Save changes</button-root>
```

## API reference

### ButtonRootElement

Custom element: `<button-root>`

| Attribute                 | Type      | Default | Description                                          |
| ------------------------- | --------- | ------- | ---------------------------------------------------- |
| `disabled`                | `boolean` | `false` | Whether the button should ignore user interaction.   |
| `focusable-when-disabled` | `boolean` | `false` | Whether the button remains focusable while disabled. |
