## Usage guidelines

- **Form controls must have an accessible name**: It can be created using a `<label>` element or the `Field` component. See the [forms guide](/react/handbook/forms).

## Anatomy

Import the helper and use it as a single part:

```ts title="Anatomy"
import { Input } from '@base-ui/lit/input';
import { html } from 'lit';

html`${Input({})}`;
```

For direct custom-element usage, import `@base-ui/lit/input` for side effects and render `<input-root>` around a native `<input>` or `<textarea>`.

### Input Data Attributes

| Attribute       | Description                                 |
| --------------- | ------------------------------------------- |
| `data-disabled` | Present when the input is disabled.         |
| `data-valid`    | Present when the input is in valid state.   |
| `data-invalid`  | Present when the input is in invalid state. |
| `data-dirty`    | Present when the input’s value has changed. |
| `data-touched`  | Present when the input has been touched.    |
| `data-filled`   | Present when the input is filled.           |
| `data-focused`  | Present when the input is focused.          |
