# Toggle Group

<Subtitle>Provides shared state to a series of toggle buttons.</Subtitle>
<Meta
  name="description"
  content="A high-quality, unstyled Lit toggle group component that provides shared state to a series of toggle buttons."
/>

## Anatomy

Import the custom element and compose it with `toggle-root` items:

```html title="Anatomy"
<toggle-group-root>
  <toggle-root value="bold">Bold</toggle-root>
  <toggle-root value="italic">Italic</toggle-root>
  <toggle-root value="underline">Underline</toggle-root>
</toggle-group-root>
```

## Examples

### Multiple

Add the `multiple` attribute to allow pressing more than one toggle at a time.

```html title="Multiple selection"
<toggle-group-root multiple>
  <toggle-root value="bold">Bold</toggle-root>
  <toggle-root value="italic">Italic</toggle-root>
  <toggle-root value="underline">Underline</toggle-root>
</toggle-group-root>
```

## API reference

### ToggleGroupRootElement

Custom element: `<toggle-group-root>`

| Attribute     | Type                         | Default        | Description                                               |
| ------------- | ---------------------------- | -------------- | --------------------------------------------------------- |
| `disabled`    | `boolean`                    | `false`        | Whether the toggle group should ignore user interaction.  |
| `multiple`    | `boolean`                    | `false`        | Whether more than one toggle can be pressed at a time.    |
| `orientation` | `'horizontal' \| 'vertical'` | `'horizontal'` | The orientation used for keyboard navigation.             |
| `loop-focus`  | `boolean`                    | `true`         | Whether focus wraps from the last item back to the first. |
