## Anatomy

Import the custom elements and assemble the parts:

```html title="Anatomy"
<meter-root value="72">
  <meter-label>Battery level</meter-label>
  <meter-track>
    <meter-indicator></meter-indicator>
  </meter-track>
  <meter-value></meter-value>
</meter-root>
```

## Usage guidelines

- Use `<meter-label>` or an explicit `aria-labelledby` reference so assistive technology announces what the value represents.
- Use `meter` for bounded measurements like capacity, storage, or rating. Use `progress` for task completion.

## API reference

### MeterRootElement

Custom element: `<meter-root>`

| Attribute / Property | Type                                | Default  | Description                                                                |
| -------------------- | ----------------------------------- | -------- | -------------------------------------------------------------------------- |
| `value`              | `number`                            | `0`      | The current value.                                                         |
| `min`                | `number`                            | `0`      | The minimum value.                                                         |
| `max`                | `number`                            | `100`    | The maximum value.                                                         |
| `.format`            | `Intl.NumberFormatOptions`          | —        | Options used to format the displayed value.                                |
| `.locale`            | `Intl.LocalesArgument`              | —        | Locale used when formatting the value.                                     |
| `.getAriaValueText`  | `(formattedValue, value) => string` | —        | Returns a human-readable value for `aria-valuetext`.                       |
| `aria-valuetext`     | `string`                            | computed | Explicit accessible value text. Overrides the computed text when provided. |

### MeterTrackElement

Custom element: `<meter-track>`

Container for the visual indicator.

### MeterIndicatorElement

Custom element: `<meter-indicator>`

Visualizes the current value as a percentage of the total range.

### MeterLabelElement

Custom element: `<meter-label>`

Accessible label for the meter.

### MeterValueElement

Custom element: `<meter-value>`

Displays the formatted value. You can customize the output with `.renderValue`.
