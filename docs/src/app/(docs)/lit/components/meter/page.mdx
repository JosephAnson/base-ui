## Anatomy

Import the custom elements and assemble their parts:

```html title="Anatomy"
<meter-root value="72">
  <meter-label>Battery level</meter-label>
  <meter-track>
    <meter-indicator></meter-indicator>
  </meter-track>
  <meter-value></meter-value>
</meter-root>
```
