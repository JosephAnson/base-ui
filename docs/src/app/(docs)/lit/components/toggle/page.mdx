## Anatomy

Import the custom element and use it as a single part:

```html title="Anatomy"
<toggle-root>Bold</toggle-root>
```

## API reference

### ToggleRootElement

Custom element: `<toggle-root>`

| Attribute         | Type      | Default      | Description                                        |
| ----------------- | --------- | ------------ | -------------------------------------------------- |
| `pressed`         | `boolean` | uncontrolled | Controlled pressed state.                          |
| `default-pressed` | `boolean` | `false`      | Initial pressed state in uncontrolled mode.        |
| `disabled`        | `boolean` | `false`      | Whether the toggle should ignore user interaction. |
| `value`           | `string`  | `undefined`  | Identifier used inside a toggle group.             |
