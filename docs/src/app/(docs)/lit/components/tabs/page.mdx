## Anatomy

Import the component and assemble its parts:

```ts title="Anatomy"
import '@base-ui/lit/tabs';
import { html } from 'lit';

html`
<tabs-root>
  <tabs-list>
    <tabs-tab value="overview">Overview</tabs-tab>
    <tabs-tab value="projects">Projects</tabs-tab>
    <tabs-tab value="account">Account</tabs-tab>
    <tabs-indicator></tabs-indicator>
  </tabs-list>
  <tabs-panel value="overview">Overview content.</tabs-panel>
  <tabs-panel value="projects">Projects content.</tabs-panel>
  <tabs-panel value="account">Account content.</tabs-panel>
</tabs-root>
`;
```

## Examples

### Links

Render a custom anchor element through the `render` prop when a tab should navigate.

```ts title="Tabs as links" {6-8}
import '@base-ui/lit/tabs';
import { html } from 'lit';

html`
<tabs-root>
  <tabs-list>
    <tabs-tab .render=${html`<a href="/overview">Overview</a>`} value="overview"></tabs-tab>
  </tabs-list>
  <tabs-panel value="overview">Overview content.</tabs-panel>
</tabs-root>
`;
```
