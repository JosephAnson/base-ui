## API Reference

### Root

A container for grouping a set of controls, such as buttons, toggle groups, or menus.
Renders a `<toolbar-root>` custom element.

**Root Props:**

| Prop        | Type                                                                          | Default        | Description                                                                                                                                                  |
| :---------- | :---------------------------------------------------------------------------- | :------------- | :----------------------------------------------------------------------------------------------------------------------------------------------------------- |
| loopFocus   | `boolean`                                                                     | `true`         | If `true`, using keyboard navigation wraps focus to the opposite end.                                                                                        |
| disabled    | `boolean`                                                                     | `false`        | Whether the component should ignore user interaction.                                                                                                        |
| orientation | `ToolbarOrientation`                                                          | `'horizontal'` | The component orientation.                                                                                                                                   |
| render      | `((props: HTMLProps, state: Root.State) => TemplateResult) \| TemplateResult` | -              | Allows you to replace the component's HTML element with a custom template.&#xA;Accepts a `TemplateResult` or a function that returns the template to render. |
