# CSPProvider

<Subtitle>Provides a CSP nonce for inline styles and controls style element injection.</Subtitle>
<Meta
  name="description"
  content="A provider element for Content Security Policy nonce support in Base UI Lit components."
/>

## Usage

```html title="Anatomy"
<csp-provider nonce="abc123">
  <!-- Components inside will use this nonce for inline styles -->
</csp-provider>
```

To disable style element injection entirely:

```html title="Disabling style injection"
<csp-provider disable-style-elements>
  <!-- No inline <style> tags will be injected -->
</csp-provider>
```

### Reading context

```ts title="Using getCSPContext"
import { getCSPContext } from '@base-ui/lit/csp-provider';

const { nonce, disableStyleElements } = getCSPContext(element);
```

## API

### `CSPProviderElement`

Custom element: `<csp-provider>`

| Attribute | Type | Default | Description |
| --- | --- | --- | --- |
| `nonce` | `string` | `''` | CSP nonce for inline styles. |
| `disable-style-elements` | `boolean` | `false` | Whether to disable `<style>` tag injection. |

### `getCSPContext(element: Element): CSPContextValue`

Returns `{ nonce, disableStyleElements }` from the nearest `<csp-provider>` ancestor.
