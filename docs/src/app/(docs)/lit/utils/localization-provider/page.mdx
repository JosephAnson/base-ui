# LocalizationProvider

<Subtitle>Defines the temporal locale provider for Base UI temporal components.</Subtitle>
<Meta
  name="description"
  content="A provider element for supplying date-fns Locale or compatible locale to Base UI Lit temporal components."
/>

## Usage

```html title="Anatomy"
<localization-provider>
  <calendar-root>...</calendar-root>
</localization-provider>
```

Set the locale programmatically:

```ts title="Setting locale"
import { fr } from 'date-fns/locale/fr';

const provider = document.querySelector('localization-provider');
provider.temporalLocale = fr;
```

### Reading context

```ts title="Using getLocalizationContext"
import { getLocalizationContext } from '@base-ui/lit/localization-provider';

const { temporalLocale } = getLocalizationContext(element);
```

## API

### `LocalizationProviderElement`

Custom element: `<localization-provider>`

| Property         | Type     | Default     | Description                               |
| ---------------- | -------- | ----------- | ----------------------------------------- |
| `temporalLocale` | `object` | `undefined` | The locale to use in temporal components. |

### `getLocalizationContext(element: Element): LocalizationContext`

Returns `{ temporalLocale }` from the nearest `<localization-provider>` ancestor.
