## Usage

```ts title="Creating the adapter"
import { TemporalAdapterLuxon } from '@base-ui/lit/temporal-adapter-luxon';

const adapter = new TemporalAdapterLuxon();

// With a custom locale:
const frenchAdapter = new TemporalAdapterLuxon({ locale: 'fr' });
```

### With the provider

```ts title="Providing to components"
const provider = document.querySelector('temporal-adapter-provider');
provider.adapter = new TemporalAdapterLuxon();
```

## API

### `TemporalAdapterLuxon`

Implements the full `TemporalAdapter` interface using Luxon.

| Option | Type | Default | Description |
| --- | --- | --- | --- |
| `locale` | `string` | `'en-US'` | The locale code for formatting and parsing. |

Supports timezone-aware dates via Luxon's built-in zone handling.
