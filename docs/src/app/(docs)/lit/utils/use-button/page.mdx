# useButton

<Subtitle>Shared button behavior utilities for any element.</Subtitle>
<Meta
  name="description"
  content="Utilities for applying button keyboard, mouse, and ARIA behavior to arbitrary elements in Base UI Lit."
/>

## Usage

```ts title="Applying button behavior"
import { applyButtonBehavior } from '@base-ui/lit/use-button';

const cleanup = applyButtonBehavior(element, {
  disabled: false,
  focusableWhenDisabled: false,
  onAction: (event) => console.log('activated'),
});

// Later, to remove all listeners:
cleanup();
```

For updating attributes without reapplying listeners:

```ts title="Syncing attributes"
import { syncButtonAttributes } from '@base-ui/lit/use-button';

syncButtonAttributes(element, { disabled: true, focusableWhenDisabled: true });
```

## API

### `applyButtonBehavior(element, options): () => void`

Applies button-like keyboard, mouse, and ARIA behavior to any element. Returns a cleanup function.

| Option | Type | Default | Description |
| --- | --- | --- | --- |
| `disabled` | `boolean` | `false` | Whether the button is disabled. |
| `focusableWhenDisabled` | `boolean` | `false` | Whether to keep focus when disabled. |
| `tabIndex` | `number` | `0` | Tab index override. |
| `native` | `boolean` | auto-detected | Whether the element is a native `<button>`. |
| `onAction` | `(event) => void` | — | Callback on activation. |

### `syncButtonAttributes(element, options): void`

Updates ARIA and DOM attributes for button behavior without reapplying listeners.

### `removeButtonBehavior(element): void`

Removes all button-related attributes from an element.
