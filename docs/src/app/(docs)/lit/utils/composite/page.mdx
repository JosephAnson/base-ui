## Usage

The composite utilities provide list and grid keyboard navigation with disabled item skipping, loop/wrap, RTL support, and Home/End keys.

```ts title="List navigation"
import { navigateList } from '@base-ui/lit/composite';

const nextIndex = navigateList({
  currentIndex: highlightedIndex,
  items: listElements,
  event: keyboardEvent,
  orientation: 'vertical',
  loop: true,
});

if (nextIndex !== -1) {
  highlightedIndex = nextIndex;
}
```

```ts title="Grid navigation"
import { navigateGrid } from '@base-ui/lit/composite';

const nextIndex = navigateGrid({
  currentIndex: highlightedIndex,
  items: gridElements,
  event: keyboardEvent,
  cols: 3,
  loop: true,
});
```

## API

### `navigateList(options): number`

Computes the next highlighted index for a 1D list. Returns `-1` if the event doesn't match a navigation key.

### `navigateGrid(options): number`

Computes the next highlighted index for a 2D grid. Returns `-1` if the event doesn't match a navigation key.

### `findNonDisabledIndex(items, options): number`

Walks forward or backward through items to find the next non-disabled index.

### `getMinListIndex(items, disabledIndices?): number`

Returns the first non-disabled index.

### `getMaxListIndex(items, disabledIndices?): number`

Returns the last non-disabled index.

### `isListIndexDisabled(items, index, disabledIndices?): boolean`

Checks whether an item should be treated as disabled.
